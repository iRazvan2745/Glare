title: Getting Started
description: Run Glare locally with server, web, database, and worker.
---

## Prerequisites

- Bun `1.3.4+`
- PostgreSQL
- Rust toolchain (for local worker builds)
- `rustic` and `rclone` binaries available to workers (or use worker container image)

## Install

```bash
bun install
```

## Configure Environment

### Web/API (`apps/web/.env.local`)

Required:

- `DATABASE_URL`
- `BETTER_AUTH_SECRET`
- `NEXT_APP_URL=http://localhost:3002`
- `APP_URL=http://localhost:3002`

## Initialize Database

```bash
bun run db:push
```

## Start Apps

```bash
bun run dev
```

## Start a Worker

```bash
worker \
  --master-api-endpoint http://localhost:3002 \
  --local-api-endpoint http://127.0.0.1:3001 \
  --api-token <worker-sync-token>
```

Get the sync token when creating the worker in UI/API.

## Local Ports

- Web + API: `3002`
- Jobs service: no public HTTP port
- Worker API: `3001` (example)
